# Projet-API

# Clonage du dépot
Entrer la commande ci dessous pour cloner le dépot dans le répertoire que vous désirez
```bash
git clone https://github.com/Soupexo/Projet-API.git
```
---

# Prérequis
- Python 3.7 ou version supérieure installé.
- Un IDE capable d’exécuter du Python (EduPython, Thonny, VS Code, PyCharm, etc.).
- Le dossier du projet déjà cloné ou dézippé.
